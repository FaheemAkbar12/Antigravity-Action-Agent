# COMPLETE CHALLENGE 1 PROJECT

## Backend

cd backend

pip install -r requirements.txt

uvicorn main:app --reload

Backend runs at:
http://127.0.0.1:8000

## Frontend

cd frontend

flutter pub get

flutter run
